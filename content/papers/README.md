# Papers

Deep dive analyses of seminal and important reinforcement learning papers.

## Purpose

Each paper analysis provides:
- Historical context and the problem being solved
- Step-by-step breakdown of the method
- Key insights and contributions
- Critical analysis and limitations
- Modern relevance and influence

## Structure

```
papers/
├── XXXX-paper-slug/
│   ├── index.mdx      # Main analysis
│   ├── prompt.md      # AI generation prompt
│   ├── summary.md     # One-page summary
│   └── assets/        # Figures, diagrams
```
